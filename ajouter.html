Ajouter un document

Remplissez le formulaire ci-dessous — aucun code n'est nécessaire. À la fin, vous téléchargez un fichier catalogue.json mis à jour que vous déposez sur GitHub.

1

Déposez le PDF

Placez votre fichier PDF dans le dossier pdfs/ du dépôt (ou utilisez un lien web).

2

Remplissez la fiche

Type, titre, auteurs, année… puis « Ajouter à la liste ».

3

Téléchargez le catalogue

Cliquez sur « Télécharger catalogue.json ».

4

Déposez sur GitHub

Remplacez le fichier catalogue.json (à la racine du dépôt) par celui téléchargé.

Déposez d'abord le PDF dans le dossier pdfs/, puis écrivez ici son nom exact.
Collez l'adresse web du document (PDF en ligne).
