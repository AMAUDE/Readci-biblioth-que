# 📚 Bibliothèque Numérique de Recherche — Côte d'Ivoire

Plateforme web libre pour **consulter et télécharger** les travaux scientifiques ivoiriens :
**livres, articles, mémoires et thèses**.

Elle est pré-alimentée avec **305 articles réels** issus de deux revues de l'Institut de
Géographie Tropicale (Université Félix Houphouët-Boigny, Abidjan) :

- **GÉOTROPE** — Revue de Géographie Tropicale et d'Environnement (112 articles, 2023–2025)
- **RETSSA** — Revue Espace, Territoires, Sociétés et Santé (193 articles, 2018–2023)

## 🗂️ Contenu du dépôt
| Élément | Rôle |
|---|---|
| `index.html` | La bibliothèque (recherche, filtres, consultation) |
| `ajouter.html` | Outil **sans code** pour ajouter un document |
| `catalogue.json` (racine) | La liste de tous les documents |
| `pdfs/` | Vos fichiers PDF |
| `GUIDE.md` | **Mode d'emploi complet en français** |

## 🚀 Mise en ligne
Ce site fonctionne sur **GitHub Pages** (gratuit). Voir le fichier **`GUIDE.md`**, section « Mettre le site en ligne ».

## ➕ Ajouter un document (sans coder)
1. Déposez le PDF dans le dossier `pdfs/`.
2. Ouvrez la page **« Ajouter un document »** du site en ligne.
3. Remplissez la fiche, puis téléchargez le nouveau `catalogue.json`.
4. Remplacez `catalogue.json` par le fichier téléchargé.

Détails pas-à-pas dans **`GUIDE.md`**.

---
*Les droits des documents appartiennent à leurs auteurs et revues respectives. Cette plateforme facilite l'accès libre déjà offert par ces revues.*
